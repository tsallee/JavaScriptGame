Check Guess

Instructions

The objective of Mastermind is to solve the secret code in as few turns as possible. The code is composed of four colored pegs, chosen by the computer. These colored pegs may be any combination of the eight possible colors, and may include two or more of the same color.

You should attempt to match the code by placing four colored pegs onto the playing board. The computer provides feedback using the key dots. A red dot symbolizes that both the color and location of a peg is correct. A white dot symbolizes a peg of the correct color but a misplaced location. You have ten attempts to crack the code.

Controls

For the active row, click on a peg to assign it a color. After assigning all pegs a color, click the check guess button. Crack the code with as few turns as possible!
